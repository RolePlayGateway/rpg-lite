Getting Started

Requirements

  • Node.js 10 (or higher)

Quick Start

Use Node's built-in package manager to install all project dependencies: npm install

Once everything installs successfully, you can run a node: yarn start

Further documentation can be found in the official documentation library. Pull Requests always welcome, especially for first-time contributors!

If you're interested in helping others learn, join us in #mentorships:fabric.pub. We're keen on helping others grow, so we hope to see you around!
